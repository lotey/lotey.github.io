mysql-5.7.12源码编译安装

请切换到root下执行操作

  • 创建mysql组

  • 创建mysql用户，并指定属于mysql组

  • 自行下载mysql源码包文件和boost源码文件，mysql-5.7.12.tar.gz和boost_1_59_0.tar.gz(必须下载此版本，高或低都不可)
    mysql-5.7.12.tar.gz
    boost_1_59_0.tar.gz

  • 解压mysql文件到/tmp

  • 解压boost源码文件到/tmp

  • 创建编译后的目录

  • 设置编译参数

  • 开始编译，多线程编译，根据自己电脑的CPU线程数量

  • 安装

  • 复制编译后的配置文件到/etc，修改配置文件名称为my.cnf

  • 修改配置信息，大体如下

    [mysqld]
    basedir = /opt/mysql-5.7.12
    datadir = /opt/mysql-5.7.12/data
    port = 3306
    socket = /opt/mysql-5.7.12/mysql.sock
    log-error = /opt/mysql-5.7.12/data/err.log
    pid-file = /opt/mysql-5.7.12/data/mysql.pid
    character-set-server=utf8
    sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES
    [client]
    port = 3306
    socket=/opt/mysql-5.7.12/mysql.sock
    default-character-set=utf8

  • 初始化mysql，并查看生成的初始化随机密码，日志文件为/opt/mysql-5.7.12/data/err.log，位置可通过my.cnf配置

  • 安装安全证书

  • 修改mysql安装目录属主

  • 启动mysql

  • 修改mysql用户密码为root

以上-p后的内容为随机密码，可在/opt/mysql-5.7.12/data/err.log中查看

  • 将myql命令加入环境变量，方便mysql命令使用

附

  • mysql启动，停止命令
